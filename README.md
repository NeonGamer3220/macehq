## 🛠️ Fejlesztés

Ez a weboldal egy FFA (Free For All) Minecraft szerverhez lett építve:
- **HTML5** - Szemantikus szerkezet
- **CSS3** - Modern stílusok és animációk  
- **Vanilla JavaScript** - Interaktív funkciók
- **Node.js** - Helyi fejlesztői szerver

## 📄 Fájlok

- `index.html` - Fő weboldal struktúra
- `style.css` - Stílus és animációk
- `script.js` - Interaktív funkciók
- `server.js` - Helyi fejlesztői szerver
