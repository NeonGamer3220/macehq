## 🚀 Quick Start

### View the Website

1. **Online:** Visit the live website (if deployed)
2. **Local Development:**
   ```bash
   git clone https://github.com/NeonGamer3220/macehq.git
   cd macehq
   node server.js
   ```
   Then open `http://localhost:8000`

### Join the Server

1. Launch Minecraft
2. Multiplayer → Add Server
3. Server Address: `macehq.mcplay.hu`
4. Join and start your adventure!
