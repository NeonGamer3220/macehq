Egy modern, DonutSMP-inspirált Minecraft szerver weboldal a MaceHQ Network számára.

## 🌐 Szerver Információk

- **Szerver Cím:** \`macehq.mcplay.hu\`
- **Verzió:** Minecraft 1.21+
- **Állapot:** 24/7 Online

## 🎮 Jellemzők

- **Egyéni Pluginok** - Minőségi életminőség javító eszközök a legjobb SMP élményért
- **Aktív Anti-Cheat** - Halásztalan játék fejlett anti-cheat rendszerekkel
- **Heti Események** - Versenyes események, építkezési versenyek és PvP tornák
- **Barátságos Közösség** - Aktív, kedves közösség építőkkel, ásványolókkal és kalandvágyókkal

## 📱 Weboldal Jellemzők

- **Responszív Tervezés** - Tökéletesen működik asztali és mobilon
- **Sötét Téma** - Modern Minecraft-esetétika
- **Interaktív Elemek** - Sima animációk és hover effektek
- **Egy Kattintásos IP Másolás** - Könnyű szerverbe lépés
- **Részecske Effektekkel** - Szép háttér animációk
- **Sima Görgetés** - Professzionális navigáció élmény

## 🚀 Gyors Kezdés

### Weboldal Megtekintése

1. **Online:** Látogass el az élő weboldalra (ha telepítve van)
2. **Helyi Fejlesztés:**
   ```bash
   git clone https://github.com/NeonGamer3220/macehq.git
   cd macehq
   node server.js
   ```
   Majd nyisd meg: `http://localhost:8000`

### Szerverhez Csatlakozás

1. Indítsd el a Minecraftot
2. Multiplayer → Szerver Hozzáadása
3. Szerver Cím: `macehq.mcplay.hu`
4. Csatlakozz és kezd el kalandolni!
